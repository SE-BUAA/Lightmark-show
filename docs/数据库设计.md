### 方法一：修改驱动属性（推荐）

1. 在 DBeaver 中找到你的 MySQL 连接，**右键** → **编辑连接**。
    
2. 切换到 **驱动属性** 标签页（Driver properties）。
    
3. 点击修改值为：
    
    1. **属性名**：`allowPublicKeyRetrieval` **值**：`true`
        
    2. **属性名**：`useSSL` **值**：`false`
        
4. 点击 **确定** 保存。
    
5. 点击 **测试连接**，应该可以成功。
    

### 方法二：修改连接 URL（同样效果）

1. 在连接编辑窗口中，切换到 **主要** 标签页（Main）。
    
2. 找到 **URL** 字段，手动追加参数：
    
3. jdbc:mysql://你的服务器IP:3306/lightmark?allowPublicKeyRetrieval=true&useSSL=false
    
4. （如果已存在 URL，只需在末尾添加 `?allowPublicKeyRetrieval=true&useSSL=false`；如果已有其他参数，用 `&` 连接）
    
5. 点击 **确定**，然后测试连接。

## 📌 说明

- `allowPublicKeyRetrieval=true` 允许客户端自动获取服务器的公钥。
    
- `useSSL=false` 关闭 SSL 加密

保存配置后，点击 DBeaver 的 **测试连接** 按钮，如果看到 “连接成功” 提示，你就可以正常访问 `lightmark` 数据库了。

如果仍然失败，请检查：
